This eCommerce website has multiple functionalities like adding to cart, searching the products, user registration and login, adding comments on a post, sending email via a contact and newsletter form.

This app uses the MySQL and MongoDB databases at the same time. It uses MongoDB to store the data in a cloud server while MySQL is used for local storage. The app also uses the JSON Web Token(JWT) to authenticate a user. It generates, stores and deletes a token in a cookie while Redis is used to store the credentials and token in a cache memory.

  • The app structure follows MVC architecture. The application contains seven parent directories and five subdirectories.
  • In these directories controllers, products data, database, JavaScript, HTML and CSS templates are written separately.
  • The front-end design of this application is taken from WP THEMES GRID while the backend is completely written in Golang. Backend is written in Golang and frontend is written in JavaScript, HTML and CSS.
