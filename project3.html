Covid Tracking App | HTML | CSS | Golang

  • Live Demo
  • Source code

This covid tracking app targets the US states and show the data of covid-19 on Google Map. The data will contain positive and negative cases, hospitalized and recovered patients, and the number of deaths.

The data contains different collections of covid data. These collections contain the data from different countries and their states. Although the data was not just limited to positive, negative cases, etc.

  • This app uses the JSON Web Token(JWT) to authenticate a user. It generates, stores and deletes a token in a cookie.
  • The app also uses Redis and MongoDB database which stores the data. Redis stores the data in a cache and MongoDB stores the data in different servers like AWS, Azure, and Google Cloud.
  • This software is written in Golang, HTML, and CSS. The directory structure is based on MVC model. The software contains seven directories in which Go code, Pandemic data, Database, HTML and CSS templates are written separately.
